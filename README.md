# Budget Tracker

## Description
This application is a budget tracker that allows users to add expenses and deposits to their budget even if they are offline. It is a progressive web application built with a MongoDB database.

### Technologies Used
* HTML
* CSS
* JavaScript
* Express
* MongoDB/Mongoose
